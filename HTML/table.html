There are four different inline and block elements

Inline Element Block Elenment
  • Inline elements occupy only sufficient width required.
  • Inline elements do not start in a new line
  • Inline elements allow other inline elements to sit behind.
  • Inline elements do not have top and bottom margin
  • Block Elements occupy the full width irrespective of their sufficiency.
  • Block elements always start in a line.
  • Block elements do not allow other elements to sit behind
  • Block elements have top and bottom margin.
 

    :inline-to-block { display: block;}

    :Inline to inline block {display: inline-block;}

    :Block to inline {display : inline}
